🎬 GyanBrige HLS Player

📋 Quick Start

1. Process a video: node videoProcessor.js process ./video.mp4 my-lecture

2. Start server: node videoProcessor.js serve

3. Enter the stream URL above and click "Load Stream"


Example URLs:

• Single quality: /media/my-lecture/playlist.m3u8

• Multi quality: /media/my-lecture/master.m3u8
